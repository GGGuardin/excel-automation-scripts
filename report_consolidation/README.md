# Report Consolidation

Merges multiple monthly CSV reports into one Excel file (details + summary). Saves 1–2 hours per month.

---

## 🇺🇸 English

### What it does
- Takes three monthly reports (January, February, March) and merges them into a single file.
- Cleans and standardizes data (countries, lesson types, payment columns).
- Adds a summary sheet with: total revenue, net earnings, average per lesson, top student, top country, busiest month, etc.
- Saves **1–2 hours per month** of manual consolidation.

### Why use it
- Stop copying and pasting between spreadsheets.
- Get a complete quarterly report in seconds.
- Avoid calculation errors.

### How to use
1. Download the script.
2. Place your monthly CSV files in the same folder.
3. Run the script.
4. Get your consolidated Excel file.

No technical knowledge required. Just Python installed.
