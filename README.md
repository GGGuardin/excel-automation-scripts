# excel-automation-scripts
Python scripts to clean, consolidate, and automate Excel reports. No fluff, just functional.

---

## 🇺🇸 English

### What these scripts do

- **Data cleaning** → Takes a messy sales CSV and turns it into a clean Excel file with totals, best-sellers, and a summary sheet. Saves 2–3 hours per week.
- **Report consolidation** → Merges multiple monthly CSV reports into one Excel file (details + summary). Saves 1–2 hours per month.
- **Automated reporting** → Generates a ready-to-use Excel report with colors, filters, and key metrics. Saves 3–4 hours per month.

### Why use them

- You stop copying and pasting data manually.
- You get clean reports without errors.
- You focus on your business, not on spreadsheets.

### How to use

1. Download the script you need.
2. Run it with your files.
3. Get your clean report in seconds.

No technical knowledge required. Just Python installed.
