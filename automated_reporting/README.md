# Automated Reporting

Generates a ready-to-use Excel report with colors, filters, and key metrics. Saves 3–4 hours per month.

---

## 🇺🇸 English

### What it does
- Takes a CSV file and turns it into a professional Excel report.
- Adds conditional formatting (green for positive earnings, red for zero).
- Includes filters, frozen headers, and an executive summary sheet.
- Calculates total revenue, number of lessons, average earnings, top country, etc.
- Saves **3–4 hours per month** of manual report creation.

### Why use it
- Get a presentation-ready report in seconds.
- Stop formatting spreadsheets by hand.
- Make your data easy to understand at a glance.

### How to use
1. Download the script.
2. Run it with your CSV file.
3. Get your formatted Excel report.

No technical knowledge required. Just Python installed.
