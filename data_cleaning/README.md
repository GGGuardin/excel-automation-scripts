# Data Cleaning

Takes a messy sales CSV and turns it into a clean Excel file with totals, best-sellers, and a summary sheet. Saves 2–3 hours per week.

---

## 🇺🇸 English

### What it does
- Removes empty rows and fixes incorrect values.
- Standardizes product names, payment methods, and locations.
- Adds a summary sheet with key metrics: total revenue, average ticket, best-selling item, top location, peak month, etc.
- Saves **2–3 hours per week** of manual cleaning.

### Why use it
- Stop fixing the same errors over and over.
- Get a clean report in seconds, not hours.
- Focus on your business, not on spreadsheets.

### How to use
1. Download the script.
2. Run it with your CSV file.
3. Get your clean Excel file.

No technical knowledge required. Just Python installed.
